# To-Do List Application (Django)

## 📌 Description
This is a Django-based To-Do List application that allows users to manage daily tasks efficiently.  
Users can create, edit, complete, and delete tasks through a clean and simple interface.

The project demonstrates backend development with Django, CRUD operations, MVC architecture, and proper Git/GitHub workflow.

---

## 🛠 Tech Stack
- Python
- Django
- SQLite (development)
- HTML / CSS
- Git & GitHub

---

## 🚀 Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- View pending and completed tasks
- Delete tasks
- Clean and simple UI
